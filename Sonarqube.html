Instalacion

Link pagina oficial: Sonarqube Download

Docker install: $ docker pull sonarqube

Iniciar imagen docker: $ docker run -p 9000:9000 --name sonarqube sonarqube:latest

Entrar a la interfaz despues de iniciar los servicios: http://{ IP }:9000
